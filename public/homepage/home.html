🚀 How to Start the Cyber Clash Quiz Game

  1. Run node index.js in your terminal to launch the server.
  2. Open all three links below in separate tabs:
    • 👑 Admin Panel
    • 🧑‍🎮 Player Sign-Up
    • 📺 Live Viewing Screen
  3. In Admin, click the Start button to kick off the game.
  4. On the Sign-Up page, enter your name and select your emoji avatar 🎭.
  5. Answer questions within the timer ⏳ once gameplay begins.
  6. After time’s up, check your scores and leaderboard rankings.
  7. You can also host the game on a Raspberry Pi or server to play across multiple devices.

🔥 Enjoy the game! May the best mind win.
